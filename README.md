# Wenchao Huang — Research Homepage

This is a static, English-first academic homepage for Wenchao Huang.

## Preview locally

Open `index.html` directly in a browser, or run a local server in this folder:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a public repository named `YOUR-USERNAME.github.io`.
2. Upload `index.html`, `styles.css`, and the three SVG illustrations: `embodied-ai.svg`, `autonomous-driving.svg`, and `robotic-planning.svg`.
3. In the repository, open **Settings → Pages** and deploy from the `main` branch.
4. The homepage URL will be `https://YOUR-USERNAME.github.io`.

Before publishing, replace the email address if needed and verify the publication status and affiliation details.
